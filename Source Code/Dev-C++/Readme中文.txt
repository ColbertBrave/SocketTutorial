在Dev-C++中编译
------------------------------


所有东西都包含了, 所以配置非常简单. 注意, however,如果你的目录不是C:\Dev-cpp的话你需要重新链接libws2_32.a文件 .否则你会得到一个编译错误：无法找到库文件.


在其他编译器中编译
-------------------------------
你需要重新链接libws2_32.a文件至项目，然后添加以下文件:

1. Chat1.cpp
2. Chat1.h
3. Chat1.rc

Chat1_private.h和Chat1_private.rc 只在Dev-C++中使用.
